<p>Dockerfile과 GCP를 사용한 CI/CD 자동화</p>
<p>MongoDB Atlas를 사용하여 애플리케이션과 데이터베이스 분리

<p>Express의 request 타입을 확장하여 미들웨어에서 공유 가능한 context 생성</p>
<p>S3 bucket으로 이미지 파일 저장소 분리</p>
